# Privacy Policy for Using This Toolkit

Legacy Journal is intentionally separated into **public tools** and **private human records**.

## Never commit by default

- Names of children or vulnerable people
- Home addresses or precise location patterns
- Phone numbers, email addresses, account identifiers, or credentials
- School, employer, medical provider, court, or treatment details
- Medical, legal, recovery, financial, or relationship history
- Private messages from people who did not consent to publication
- Images containing faces, documents, mail, screens, or location metadata
- Raw AI transcripts containing private context

## Safer publication pattern

1. Keep the original entry private.
2. Create a separate public adaptation.
3. Replace identities with roles.
4. Generalize dates and locations.
5. Remove unnecessary medical, legal, and financial detail.
6. Strip document metadata.
7. Search the final files for names, handles, addresses, and unique phrases.
8. Have another person or model perform an independent privacy review.
9. Publish only the generalized copy.

## Important limitation

Deleting a file from the newest commit does not erase it from earlier Git history. If private data is accidentally committed, stop publishing, rotate exposed credentials, and use Git history-rewrite procedures before resuming.
