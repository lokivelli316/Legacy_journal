## Publishing Pipeline Notes

- **Amazon:** KDP direct for Kindle, paperback, and hardcover (where page count qualifies).
- **Wide eBook:** Draft2Digital, Amazon unchecked (managed separately via KDP). $20 one-time activation fee for new accounts, $12/year maintenance fee only if net earnings stay under $100/year.
- **Wide print:** choose one — IngramSpark or Draft2Digital Print. Do not run both through the same channels; turn off KDP Expanded Distribution if another service is handling Ingram-channel print distribution.
- **ISBNs:** owned ISBNs recommended for paperback and hardcover editions to keep publisher/imprint identity consistent across pipelines. Kindle eBooks don't need one.
- **KDP Select:** stay out initially — it requires exclusivity that conflicts with wide distribution. Revisit later if wide distribution data suggests it's worth it.
- **AI disclosure:** KDP distinguishes AI-assisted (you wrote it, AI edited/organized) from AI-generated (AI produced the actual text). Evaluate each manuscript on its own creation process rather than one blanket answer for the whole catalog.
